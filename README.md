# Aizome LP

地方中小企業向け AI研修・AI導入支援サービス「Aizome」のランディングページ。

- **運営**: 株式会社 Piic
- **配信**: GitHub Pages (静的HTML)
- **お問い合わせ**: Formspree

## 構成

```
aizome-lp/
├── index.html       # シングルページLP
├── assets/          # 画像・動画
└── README.md
```

## ローカルプレビュー

```bash
python3 -m http.server 8000
# http://localhost:8000/ にアクセス
```

## デプロイ

`main` ブランチへのpushで GitHub Pages が自動更新されます。
